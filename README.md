# Welcome to develpoment branch.

create a branch with your name or any prefered phrase

use the command

```
git checkout develop
git pull
git checkout -b <name>
```

All the python servers should be wsgi
And build pattern is Factory model

start working on you branch

update .gitignore and enjoy!!!

WORKSPACE:

./auth has auth server code
./benchmarks has test reports
./confs has configurations
./dashboards has grafana dashboards
./docs has any related docs
./frontend react app for frontend
./hf_server has the code tobe deployed in hugging face
./main the core application server which makes api calls and read write data is here
./ODM has the Object Document Models for mongodb.

MOHITH works on hf_server folder (frontend if possible)

FAHAD works on main folder

NISHANTH works on auth folder
